JVM Configuration Properties

The properties listed on this page are used to configure the JVM which is launched by the Java Service Wrapper. The values set to many of these properties are dependent on the method used to integrate an application with the Wrapper. Please make sure you are familiar with the integration methods before modifying any of these properties

  • wrapper.java.command
  • wrapper.java.classpath.<n>
  • wrapper.java.library.path.<n> (2.2.9)
  • wrapper.java.library.path.append_system_path (3.1.0)
  • wrapper.java.additional.<n>
  • wrapper.java.mainclass
  • wrapper.app.parameter.<n>
  • wrapper.java.initmemory
  • wrapper.java.maxmemory
