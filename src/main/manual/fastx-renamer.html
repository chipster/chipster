Utilities / Rename reads

Description

Given a FASTQ file, this renames the read identifiers as specified by the user.

Parameters

  • Rename read identifiers to (Running number, Sequence itself) [Running number]

Details

You can use this tool for example for keeping track of the original read sequences, if you plan to trim the reads afterwards.

Output

A FASTQ file containing the renamed reads.

Reference

This tool is based on the FASTA/Q Renamer tool of the FASTX package.
