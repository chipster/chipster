CNA-seq / Filter copy number bins

Description

Filters out low-quality bins.

Parameters

  • QDNAseq blacklist (filter out, keep) [filter out]
  • ENCODE blacklist (filter out, keep) [filter out]
  • Minimum mappability (0-100) [0]
  • Sex chromosomes (filter out, keep) [filter out]

Details

This tool filters out low-quality bins from subsequent analyses. The data rows are not really removed from the data table, but only flagged to be ignored.

Output

The table from the input file with the column "use" updated for the new filtering.
