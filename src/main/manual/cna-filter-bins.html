CNA-seq / Filter copy number bins

Description

This tool takes as input a table on binned read counts (from Define CNA-seq experiment), and filters out bins that are known to behave spuriously.

Parameters

  • QDNAseq blacklist (remove, keep) [remove]
  • ENCODE blacklist (remove, keep) [remove]
  • minimum mappability (0-100) [0]
  • sex chromosomes (remove, keep) [remove]

Details

By default, bins in the QDNAseq or ENCODE blacklists are filtered out, as are the sex chromosomes. A minimum mappability can also be specified.

Output

Data table with the chosen bins filtered out.

References

Scheinin et al. (2014) DNA copy number analysis of fresh and formalin-fixed specimens by whole-genome sequencing: improved correction of systematic biases and exclusion of problematic regions. Manuscript submitted.

Mappability: Koehler et al. (2011) The uniqueome: a mappability resource for short-tag sequencing. Bioinformatics 27: 272-274
