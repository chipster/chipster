Copy number aberrations / Plot copy number profiles

Description

Plot copy number profiles of individual arrays from called aCGH data.

Parameters

  • Samples to be plotted (1...n) [1]
  • Chromosomes to be plotted (0...25) [0]
  • Resolution (0..1) [0.1]

Details

Original log ratios are shown in black, segmented values in brown, probabilities of losses in red (left scale), and probabilities of gains in blue (inverted left scale)

Output

aCGH profile of individual sample(s).

References

Wiel et al. (2007) CGHcall: calling aberrations for array CGH tumor profiles. Bioinformatics 23: 892-894
