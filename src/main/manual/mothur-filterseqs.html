Metagenomics / Filter sequence alignment with Mothur

Description

Filters out columns from a fasta formatted sequence alignment.

Parameters

None

Details

Removes columns where every character is '-'. These empty columns are not included in calculating distances because they have no information in them. By removing these columns, the calculation of a large number of distances is accelerated. Removes also columns where one or more sequences have '.'

This tool is based on the Filter.seqs command of the Mothur package.

Output

The analysis output consists of the following:

  • filtered-aligned.fasta: Filtered alignment
  • filtered-summary.tsv: Summary statistics for the filtered alignment
  • filtered-log.txt: Lists the number of columns removed
