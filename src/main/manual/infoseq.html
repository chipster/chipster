Display basic information about sequences

Description

This tool produces a table of that contains information about the sequences in the defined sequence set.

Parameters

None

Details

This tools runs EMBOSS command infoseq that displays basic information about one or more input sequences. The input sequences can be in any of the EMBOSS compatible sequence formats.

Following properties are included in the result table:

  • Sequence name
  • Sequence accession number
  • Sequence type (protein or nucleotide)
  • Sequence length
  • Organism (if the information is available in the sequence format)
  • Description (if the information is available in the sequence format)

    • Output

    Output is a text file.
