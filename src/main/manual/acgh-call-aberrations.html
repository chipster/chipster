aCGH / Call copy number aberrations

Description

Call copy number aberrations from segmented data (either aCGH log ratios, or log2-transformed NGS data).

Parameters

  • Number of copy number states (3, 4) [3]
  • Organism (human, other) [human]
  • Human genome build (GRCh37/hg19, NCBI36/hg18, NCBI35/hg17, NCBI34/hg16) [GRCh37/h19]

Details

After normalizing and segmenting the log ratios, this tool is used to obtain discrete copy number levels with CGHcall. Calling of three copy number states means separating losses, normals and gains. Amplifications can also be detected separately as the fourth copy number state. For each probe, the frequencies of arrays exhibiting a loss/gain/amplification are also included.

The input data must be segmented and contain these columns: chromosome, start, end.

Output

The inputted original and segmented log ratios are accompanied with hard copy number calls, and probabilities assigned to each call.

A plot is also returned that shows the frequencies of gains (blue) and losses (red).
