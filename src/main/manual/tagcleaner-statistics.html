Utilities / Statistics for primers/adaptors

Description

Prints the number of tag sequences matching for different numbers of mismatches.

Parameters

  • Input type (FASTQ, FASTA) [FASTQ]
  • Tag sequence at 3'-end (String) [-]
  • Tag sequence at 5'-end (String) [-]

Details

Prints the number of tag (i.e. primer/adapter) sequences matching for different numbers of mismatches.

Input can be a FASTQ file or a FASTA file.

No trimming will be performed.

Output

The output is a tab-delimited text file.

References

This tool uses the TagCleaner package. Please cite the article:

Schmieder R, Lim YW, Rohwer F, Edwards R, 2010. TagCleaner: Identification and removal of tag sequences from genomic and metagenomic datasets. BMC Bioinformatics. 2010, 11:341.
