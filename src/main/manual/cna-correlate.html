Copy number aberrations / Plot correlations of called copy number data

Description

This tool calculates the correlations of copy number aberrations between different regions the the genome and plots them.

Parameters

  • Human genome build to use (GRCh37/hg19, NCBI36/hg18, NCBI35/hg17, NCBI34/hg16) [GRCh37/hg19]

Details

Plotting the correlations can help to identify specific aberrations that often co-occur. Two gains that co-occur will have a positive correlation coefficient (plotted in blue), whereas a gain and a loss that co-occur will have a negative correlation coefficient (red).

This tool should be run after running the Identify common regions from called copy number data tool.

Output

Two plots of the correlation matrix, one showing each region with the same size, and another showing each region according to their true genomic size. Also a numberic table of the correlation coefficients is produced.
