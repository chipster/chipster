Sort table by column value

Description

This tool sorts a table based on a selected column. The sorting is done either based on numerical order or alphabetical order, depending on the data type of the column. The sorting order can be selected to be ascending or descending.

You can sort a table using several columns by running several sorting operations in cascade so that the primary sorting criteria is used in the last sorting operation.

Output

The tool produces a new file containing the sorted rows of the input table.
