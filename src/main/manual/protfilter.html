Filter protein sequence sets

Description

Select sequences from protein sequence set using different criteria.

This tool reads a set of protein sequences as an input and selects the ones that match the given filtering criteria. Filtering can be done based on:

  • Sequence length
  • Sequence mass
  • Sequence pattern
  • String in the sequence description

The input protein sequences can be in any of the EMBOSS compatible sequence formats.

Output

Output is a fasta formatted sequence file.
