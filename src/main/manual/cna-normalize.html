CNA-seq / Normalize copy number data

Description

Normalizes copy number data before segmentation.

Parameters

  • Log2 transform (yes, no) [yes]
  • Normalization method (median, mode, none) [median]
  • Number of chromosomes (...) [22]

Details

Copy number data is log2 transformed (if chosen), and then normalized using the chosen method.

The input data must contain these columns: chromosome, start, end.

Output

Normalized values.
