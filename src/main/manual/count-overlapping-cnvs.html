aCGH / Count overlapping CNVs

Description

Counts overlapping CNVs from the database of genomic variants.

Parameters

  • Human genome build to use (GRCh37/hg19, NCBI36/hg18, NCBI35/hg17) [GRCh37/hg19]

Details

The list of known copy number variations (CNVs) is downloaded from the Database of Genomic Variants. For each are of interest (probes, genes, or chromosomal regions) two values are calculated: the number of entries in the database that overlap with the area of interest, and the number of base pairs within known CNVs per megabase of sequence of the are of interest. These are reported in columns cnv.count and cnv.per.Mb, respectively.

Output

The input data table appended with columns cnv.count and cnv.per.Mb.

References

Iafrate et al. (2004) Detection of large-scale variation in the human genome. Nat Genet 36: 949-951
