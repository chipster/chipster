Preprocessing / Filter by column term

Description

Filters NGS data based on textual terms in the selected data column.

Parameters

  • Column to filter by (...) [empty]
  • Term to match (...) [empty]
  • Does the first column have a title (yes, no) [yes]

Details

Filters rows in a table based on textual terms in the selected data column. For example, you can filter for text matches such as "upstream" or "downstream" in the output file from the "Find nearest genes for regions" tool. Notice that rows where the match term is only a part of the whole term are also passing the filter.

Output

A tabular text file containing only the set of rows that pass the filtering criteria.
