Copy number aberrations / Fetch probe positions from GEO

Description

Fetches microarray probe positions from the GEO database.

Parameters

  • Accession number of the platform in question [empty]

Details

The probe positions are fetched from the GEO database and appended to the input file. For a list of available platforms, please see this list. Please see the GEO entry for the particular platform to very which genome build the mappings refer to.

Output

The input file appended with three new columns: chromosome, start, end.

References

GEO:
Edgar et al. (2002) Gene Expression Omnibus: NCBI gene expression and hybridization array data repository. Nucleic Acids Res 30: 207

Ensembl:
Flicek et al. (2010) Ensembl's 10th year. Nucleic Acids Res 38: D557-62
