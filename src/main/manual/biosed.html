Replace or delete sequence section

Description

This tool is based on EMBOSS command bioseq. This tool searches for a target subsequence in one or more input sequences and replaces matching region with an insert subsequence, or optionally, just deletes the target subsequence where found. If the target subsequence occurs more than once, then each instance of the target is replaced or deleted.

The input sequences can be in any of the EMBOSS compatible sequence formats.

For more details, please check the manual page of biosed command.

Output

Output is a PNG, SVG or PDF formatted image file.
