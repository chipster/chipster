Metagenomics / Merge FASTA or QUAL files

Description

Merge two or more FASTA or QUAL files together.

Parameters

File type (FASTA, QUAL) [FASTA]

Details

Merges two or more FASTA or QUAL files together. You should take care to only merge files of the same type. The File type parameter sets the file name of the output.

Output

The analysis output consists of the following:

  • merged.fasta: Merged FASTA files
  • merged.qual: Merger QUAL files
