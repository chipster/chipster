Chipster manual

Chipster is a versatile data analysis platform with interactive visualizations and workflows. It offers a comprehensive collection of analysis tools for next generation sequencing (NGS), microarray and proteomics data. The NGS functionality covers analysis from quality control and alignment to downstream applications such as pathway analysis and motif detection (please see the analysis tool summary for details), and more analysis tools are added all the time. The built-in Chipster genome browser allows users visualize reads and results in their genomic context. The microarray functionality covers expression, aCGH, miRNA, methylation and SNP arrays, and allows users to integrate expression data with miRNA and aCGH data.
