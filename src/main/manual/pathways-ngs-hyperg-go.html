NGS ChIP-seq / GO enrichment for list of genes

Description

Performs a statistical test for enrichment of GO terms in a query list of genes. The input file should come from the tool Find unique and annotated genes.

Parameters

  • Ontology category (all, biological process, molecular function, cellular component) [biological process]
  • P-value threshold (0-1) [0.05]
  • Minimum population (1-1000000) [2]
  • Conditional testing (yes, no) [yes]
  • P-value adjustment method (none, BH, BY) [none]
  • Search for over or under-represented genes (under, over) [over]

Details

This tool takes a list of Entrez gene-ID:s, like the unique-genes-list.tsv output file produced by the "Find nearest genes for regions" tool. A hypergeometric test is used to check for over-represented (or under-represented) GO categories relative to the whole genome and the nalysis can be restricted to GO terms which have a minimum, user-specified number of genes mapped to them (by default 2).

The default behavior is that in order to avoid the identification of directly related GO terms with considerable overlap of genes, this tool uses a method which conditions on all child terms that are themselves significant at a specified p- value cutoff. First the leaves of the graph (those terms with no child terms) are tested. All genes annotated at significant children are then removed before testing the parent terms. This continues until all terms have been tested. It is also possible to run the tests unconditional by setting the appropriate parameter. In that case it is also possible to apply multiple testing correction.

Output

Output is a text file and an HTML file with a list of significant categories.
