aCGH / Plot copy number profiles from called aCGH data

Description

Plot copy number profiles of individual arrays from called aCGH data.

Parameters

  • Samples to be plotted (1...n) [1]
  • Chromosomes to be plotted (0...25) [0]
  • Image width (200...3200) [600]
  • Image height (200...3200) [600]

Details

Original log ratios are shown in black, segmented values in blue, probabilities of losses in red (left scale), and probabilities of gains in green (inverted left scale)

Output

aCGH profile of individual sample(s).

References

Wiel et al. (2007) CGHcall: calling aberrations for array CGH tumor profiles. Bioinformatics 23: 892-894
