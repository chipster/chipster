Annotation / miRNA annotations

Description

Annotates the miRNA probes, and creates a web-page with links to target information as predicted by TargetScan and MIRANDA.

Parameters

none

Details

Annotates miRNAs which have identifiers recognized by the miRNABase database (such as hsa-let-7a or MIMAT0004481).

Output

Tool writes an HTML and a text file with annotations for the genes.
