Copy number aberrations / Count the number of aberrations per sample

Description

For each sample, counts the number of aberrations that are at least as big as the minimum size provided.

Parameters

  • Minimum size in base pairs(...) [0]

Details

This tool calculates the number of distinct copy number aberrations for each sample. The user can specify the minimum size that should be counted. Different aberration types (gains, losses, and if identified separately during calling, also amplifications and homozygous deletions) are counted separately and also a total provided. Also the minimum, maximum, and average number of aberrations is provided per sample.

Output

A table of aberration counts.
