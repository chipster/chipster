Metagenomics / Summarize sequences with Mothur

Description

Summarize the quality of sequences in an unaligned or aligned fasta-formatted sequence file.

Parameters

Details

This tools summarizes the quality of sequences in an unaligned or aligned fasta-formatted sequence file. It lists sequence start and end coordinates, sequence length, number of ambiguous sequences, the longest homopolymer length. It also calculates summary statistics for all the sequences.

This tool is based on the Mothur package.

Output

The analysis output consists of the following:

  • summary.tsv: Lists the values for each sequence, one sequence per line
  • log.tsv: Lists summary statistics for all the sequences
