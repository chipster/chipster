Utilities / Extract samples from dataset

Description

Creates a subset of samples

Parameters

  • Which column contains the extraction information

Details

You need to create a new column in the phenodata file where you specify which samples need to be kept. Mark these samples with 1 and the others with 0. You can use this tool for normalized microarray data as well as NGS count tables.

Output

A new dataset with only the extracted samples in a tab-delimited text file, and a cropped phenodata that corresponds to the extracted dataset.
