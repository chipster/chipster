Utilities / Preprocess count table

Description

Converts count table to Chipster format (by adding "chip." to the count column names) and generates the phenodata file.

Parameters

  • Experiment type [empty] (You can define experiment type, for example rna-seq)
  • Keep or discard annotation columns after preprocessing [yes, no] (no)

Details

You need to import your count table to Chipster using the Import tool. Make sure to mark every count column as Sample and the column containing feature names as Identifier.

Output

Count table and phenodata file.
