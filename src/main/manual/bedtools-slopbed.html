BEDTools slopBed

Description

Add requested base pairs of "slop" to each feature.

Parameters

  • The number of base pairs to subtract from the start coordinate (Integer) [0]
  • The number of base pairs to add to the end coordinate (Integer) [0]
  • Define additions based on strand (yes, no) [no]
  • Define additions as a fraction of the feature's length (yes, no) [no]
  • The fraction to subtract from the start coordinate (Decimal) [0.0]
  • The fraction to add to the end coordinate (Decimal) [0.0]

Details

slopBed will increase the size of each feature in a feature file be a user-defined number of bases. SlopBed will restrict the resizing to the size of the chromosome (i.e. no start < 0 and no end > chromosome size).

Output

The output is a BED file.
