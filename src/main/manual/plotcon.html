Alignment conservation plot

Description

Plot conservation of a sequence alignment.

This tool uses EMBOSS command plotcon to draws a conservation plot of a sequence alignment.

The input sequences can be in any of the EMBOSS compatible sequence alignment formats. Note that this tool reads just one input file, that should contain all the sequences to be analyzed.

For more details, please check the manual pages of plotcon command.

Output

Output is a fasta formatted sequence file.
