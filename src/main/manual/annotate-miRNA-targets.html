Annotation / Find miRNA targets

Description

Fetches predicted gene targets from a miRNA selected database.

Parameters

  • species (human) [human]
  • database (miranda, mirbase, mirtarget2, pictar, tarbase, targetscan) [mirtarget2]

Details

Given a list of miRNA which have identifiers recognized by the miRNA database (such as hsa-let-7a or MIMAT0004481), this tool searches one of the specified databases to fetch the predicted gene targets.

Output

A table listing the miRNAs and their predicted target genes. The first column contains as identifiers miRNA-target pairs, which allows you to intersect result lists from different databases through the interactive Venn diagram visualization. Genes are listed with an Entrez gene identifiers, gene symbol, and a description obtained from the Ensembl database.
