Primer

Description

Pick PCR primers and hybridization oligos.

This tool runs EMBOSS command eprimer3 that uses primer3 program from the Whitehead Institute.

This tool picks primers for PCR reactions, considering as criteria:

  • oligonucleotide melting temperature, size, GC content, and primer-dimer possibilities
  • PCR product size
  • positional constraints within the source sequence
  • miscellaneous other constraints
All of these criteria are user-specifiable as constraints.
eprimer3 can also pick hybridisation oligos that are internal to the product.

The input sequences can be in any of the EMBOSS compatible sequence formats.

For more details, please check the manual page of eprimer3 command.

Output

Output is a text file.
