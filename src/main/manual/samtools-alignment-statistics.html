Count alignment statistics for BAM

Description

Given a BAM file, this tool counts statistics for alignments.

Parameters

None

Details

Counts statistics for alignments in a BAM file. This tool is based on the SAMtools flagstat tool.

Output

Output is a txt file containing the statistics.

References

This tool is based on the SAMtools package.
