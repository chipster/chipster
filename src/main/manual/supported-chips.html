Supported chip types

Currently the following chip types have annotation packages in Chipster, which means that they can be annotated and used for pathway and promoter analysis. Please note that if your array doesn't appear on the list, you can still bring the data into Chipster and run preprocessing, statistical analysis and clustering for it. The only exception is Affymetrix data, which requires the CDF and probe packages for the preprocessing to work. Currently some Affymetrix chip types (marked with asterisk on the list) have only the CDF and probe packages in Chipster, which means that the annotation based analyses are not available for them. The names in brackets indicate the annotation package names, which you need to use (with the ending .db) to define the chiptype if bringing normalized data into Chipster.
