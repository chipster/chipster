Copy number aberrations / Plot copy number aberration frequencies

Description

Plot copy number aberration frequencies for different groups.

Parameters

  • Phenodata column separating the samples into two or more groups
  • Chromosomes to be plotted (0...25) [0]

Details

This tools makes a frequency plot of copy number aberrations in each group defined by the selected phenodata column.

Output

Plots for each group showing the frequencies of gains (blue) and losses (red).
