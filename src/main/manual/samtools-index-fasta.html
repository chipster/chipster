Utilities / Index FASTA

Description

Given a FASTA file, this tool creates an index file (.fai) for it.

Parameters

None

Details

Given a FASTA file, this tool creates an index file (.fai) for it. Index file is needed to view a sequence in the Chipster genome browser.

Output

Output is an index file.

References

This tool is based on the SAMtools package.
