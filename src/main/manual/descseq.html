Modify sequence name or description

Description

Extract regions from a sequence.

This tool uses EMBOSS command descseq to alter the name and / or description of a sequence. All other records including the sequence itself are left unaltered.

The input sequences can be in any of the EMBOSS compatible sequence formats.

For more details, please check the manual page of descseq command.

Output

Output is a PNG, SVG or PDF formatted image file.
