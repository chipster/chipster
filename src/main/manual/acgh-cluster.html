aCGH tools / Cluster called copy number data

Description

Perform clustering of aCGH arrays.

Parameters

  • Type of copy number calls to use (soft, hard) [soft]
  • Phenodata column separating the samples into two or more groups

Details

Clustering of aCGH arrays can be performed using either soft or hard calls. Hard calls divide the data into either losses, normals, or gains, whereas soft calls refer to the respective probabilities of these calls. The preferred choice is to use soft calls whenever they are available.

Output

An image showing the clustering of the arrays.

References

Wieringen et al. (2008) Weighted clustering of called array CGH data. Biostatistics 9: 484-500
