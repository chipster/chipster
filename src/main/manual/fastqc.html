Quality control / Read quality with FastQC

Description

Given a FASTQ or BAM file, this tool produces several quality control plots for the reads.

Parameters

  • File type (bam, fastq) [fastq]

Details

FastQC supports all quality encoding variants of FASTQ files and BAM files. GZip compressed files are automatically detected and unzipped. For a detailed description of the plots, please see the FastQC manual.

Output

A PDF file with plots and tables produced by the analysis modules.

Reference

This tool is based on the FastQC package by Simon Andrews et al.
