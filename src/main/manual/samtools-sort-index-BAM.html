Utilities / Sort and index BAM

Description

Given a BAM file, this tool sorts it and creates an index file (.bai).

Parameters

None

Details

This tool is similar to the preprocessing tool which you are prompted to use when importing BAM files to Chipster. The preprocessing tool is based on the (pickier) Picard package and runs on your own computer, while this one is based on the more relaxed SAMtools package and runs on the server.

Output

Output is a sorted BAM file and an index file.

References

This tool is based on the SAMtools package.
