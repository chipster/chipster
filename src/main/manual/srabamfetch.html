Retrieve read alignments from SRA database

Description

This tool retrieves read alignments from the SRA database based on the SRR ID of a dataset. The data retrieval and processing is done with the SRA Toolkit package.

Details

As the SRA archive files can be very large, downloading the data can take a long time.

Sample entry names: 

SRR412821
ERR431372

Output

The mapping results of the defined SRA experiment are imported to Chipster in BAM format. You also print out just the primary alignments or add the unaligned reads to the output
