Analysis tools for next generation sequencing data

Chipster's NGS analysis tools are grouped in the categories listed below. Each analysis tool has its own detailed manual page, which consists of four sections: General description, parameters, details, output and reference. The parameter section contains the list of parameters, their possible values (1-100) and their default values [10]. The manual for Chipster's built-in genome browser can be found here.
