miRNA-seq / Find miRNA targets

Description

Fetches predicted gene targets from a selected database given a list of miRNA:s.

Parameters

  • species (human, mouse, rat) [human]
  • database (miranda) mirbase, mirtarget2, pictar, tarbase, targetscan) [mirtarget2]

Details

Given a list of miRNA:s, which have identifiers recognized by the miRNA database (such as hsa-let-7a or MIMAT0004481), this tool searches one of the specified databases to fetch the predicted gene targets. Currently, the following databases are supported: miRanda, miRbase, miRtarget2, PicTar, TarBase or targetScan.

Output

A table listing the miRNA:s in one column and their predicted target genes in another. The sites column corresponds to the number of times each miRNA-gene -pair has been listed in the given database. In most cases, this equals to the number of putative miRNA sites within the given Entrez gene.
