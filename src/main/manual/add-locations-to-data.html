Copy number aberrations / Cluster called copy number data

Description

Annotation / Add genomic location to data

Description

Annotates genes with chromosomal location information, and adds the results to the datafile. The result file can be visualized using the Genome browser.

Parameters

  • Annotate using (gene symbols, probe IDs) [gene symbols]
  • Species (human, mouse, rat) [human]

Output

A .tsv file similar to the input file, with chromosomal location information added.
