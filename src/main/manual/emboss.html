Other sequence utilities

Description

This tool runs one of the following EMBOSS sequence analysis commands for the input data set. The selected command is run with default parameters.

One of the following commands can be selected:

  • chips Calculate Nc codon usage statistic
  • degapseq Remove non-alphabetic e.g. gap characters from sequence
  • geecee Calculate fractional GC content of nucleic acid sequences
  • helixturnhelix Identify nucleic acid-binding motifs in protein sequences
  • maskambignuc Mask all ambiguity characters in nucleotide sequences with N
  • maskambigprot Mask all ambiguity characters in protein sequences with X
  • prettyseq Write a nucleotide sequence and its translation to file
  • showseq Display nucleic sequences with features in pretty format
  • showpep Display protein sequences with features
  • showfeat Display features of a sequence in pretty format
  • seqcount Count sequences
  • sizeseq Sort sequences by size

Output

Output is sequence or report file.
