Reverse sequence

Description

Reverse and complement a nucleotide sequence.

Parameters

  • Reverse sequence Set this to be false if you do not wish to reverse the output sequence.
  • Complement sequence Set this to be false if you do not wish to complement the output sequence.

    • Details

    This tools runs EMBOSS command revseq to convert the given sequence, into its reverse and/or complement sequence. The input sequences can be in any of the EMBOSS compatible sequence formats.

    Output

    Output is a sequence file.
