Normalisation / Affymetrix exon arrays

Description

Preprocesses Affymetrix exon array CEL-files to give expression estimates and call values for genes.

Parameters

  • Chiptype (empty, human, mouse, rat) [empty]
  • Summary feature (gene, exon) [gene]

Details

Computes the expression values for genes using the RMA (robust multiarray) method.

The correct chiptype is automatically selected, when the user defines the species.

Expression estimate can be created for each exon or for each gene. CDF files for exon summaries were download from Manchester University, and genes are summarized using the CDF files from University of Michigan.

Output

A tab-delimited text file containing gene names and expression estimates. This file is suitable for all further analyses.
