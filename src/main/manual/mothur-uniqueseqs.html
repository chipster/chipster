Metagenomics / Extract unique aligned sequences with Mothur

Description

Given a fasta-formatted alignment and a names file, removes identical sequences from the alignment.

Parameters

None

Details

When you filter an alignment of unique sequences, you might be removing bases that accounted for differences between the sequences. It is therefore recommended to remove redundant sequences from the filtered alignment in order to speed up distance calculations later. In addition to the alignment, you need to supply the names file that was created by the tool "Trim and filter reads with Mothur". When redundant sequences are found, the list of names corresponding to the sequence names will be merged.

This tool is based on the Unique.seqs command of the Mothur package.

Output

The analysis output consists of the following:

  • unique.fasta: Alignment with redundant sequences removed
  • unique-summary.tsv: Summary statistics for the alignment
  • unique.names: A list of the unique sequences and the redundant sequences corresponding to each.
