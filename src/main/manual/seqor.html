Merge two sequence sets using logical operators.

Description

This tool joins two sequence sets with the selected logical operator. Note that this tool joins the two sequence sets based only on the sequence data. Sequence names or ID:s don't influence to the comparison.

The following logical operators combine the sequences in the following ways.

  • OR - gives all that occur in one set or the other
  • AND - gives only those which occur in both sets
  • XOR - gives those which only occur in one set or the other, but not in both
  • NOT - gives those which occur in the first set except for those that also occur in the second

The input sequences can be in any of the EMBOSS compatible sequence formats.

Output

Output is a fasta formatted sequence file.
