Chipster doesn't start on my Mac, what to do?

If Chipster doesn't launch by clicking on the launch link on the website, please try the following:
  1. Save the Java web start file chipster.jnlp on your computer by right-clicking the "Launch Chipster" link and selecting the option "Save Link As…"
  2. Usually, Chipster can be started by double-clicking the Web start file. If you get an error message like this or if the file opens only as a text file, you have to do the following:
    Right-click the file icon, go to "Open with" and choose "Java Web Start.app"
