Preprocessing / Filter by column value

Description

Filters rows in a table containing NGS based on the values in the selected data column.

Parameters

  • Column to filter by (...) [empty]
  • Cutoff (-100000...100000) [1]
  • Filtering criteria (equal-to, smaller-than, larger-than, within, outside) [smaller-than]
  • Does the first column have a title (yes, no) [yes]

    Details

    Filters rows in a table based on continuous values in the selected data column, for example genomic regions detected by the tool "Find peaks using MACS, treatment vs. control""). The options "within" and "outside" can be used to filter based on a symmetrical range of values.

    Output

    A tabular text file containing only the rows that pass the filtering criteria.
