Restrict

Description

This tool reports restriction enzyme cleavage sites in a nucleotide sequence.

The cleavage sites are determined with EMBOSS command restrict that scans one or more nucleotide sequences for cut sites for a supplied set of restriction enzymes. One or more restriction enzymes can be specified or alternatively all the enzymes in the REBASE database can be investigated. The minimum length of a recognition site to be reported must be specified. The output file shows the location of the cut sites. There are several options to control exactly what sites are reported and the format of the output file. Optionally, the fragment lengths of the forward sense strand produced by complete restriction by each restriction enzyme on its own, or by using all of the input enzymes together, may be reported. Results are added to the tail section of the report.

The input sequences can be in any of the EMBOSS compatible sequence formats.

For more details, please check the manual page of Restrict command.

Output

Output is a table or a text file.
