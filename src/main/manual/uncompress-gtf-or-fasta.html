Uncompress a file

Description

Uncompress a GTF or a FASTA file.

Parameters

  • File type (GTF, FASTA) [GTF]

Details

This tools uncompresses a zip or gzip compressed GTF or a FASTA file.

Most Chipster tools accept compressed files directly, but some tools, mainly Genome Browser, require uncompressed files. Uncompressing also allows you to visually inspect your files.

Output

The uncompressed file.
