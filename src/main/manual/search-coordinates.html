Utilities / Search by genomic coordinates

Description

Search by chromosome name and starting and ending base pair positions. The data must contain the corresponding columns.

Parameters

  • position
  • include.partial.overlaps (yes, no) [yes]

Details

Searches for features overlapping with the user-specified area in the genome. The parameter position must contain three values that are separated by tabs, hyphens, colons or two dots (e.g. X:100-200 or 7:600..700).

Output

Returns a tab-delimited text file containing the matching features.
