Copy number aberrations / Plot survival curves for called copy number data

Description

Plot Kaplan-Meier survival curves for called copy number data.

Parameters

  • Phenodata column with survival data
  • Phenodata column with survival status (1=dead, 0=alive)
  • Aberrations (gains, losses, both) [both]
  • Plot 95% confidence intervals (yes, no) [no]

Details

Plot Kaplan-Meier survival curves for each copy number region in the input file. The recommended input is output from the aCGH / Survival test for called copy number data tool after being filtered for p-value and/or false discovery rate.

Output

A Kaplan-Meier survival curve for each copy number region in the input data.

References

Wiel et al. (2007) CGHcall: calling aberrations for array CGH tumor profiles. Bioinformatics 23: 892-894
