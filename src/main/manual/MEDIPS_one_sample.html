MEDIPS - methylation analysis, treatment only

Description

This tool performs methylation analysis of sequencing data for a single file or a set of files if they are derived from a single condition, e.g., if all files are from a control experiment or a single treatment. Currently, the tool works only with human data.

Parameters

  • Species (human) [human]
  • Use only promoters (yes, no) [no]
  • Resolution for coverage calculation (25,50,100,200) [50]
  • Smoothing extension (200,400,800,1200) [400]
  • Fragment length (400,800,1600,2400) [800]
  • Save a BED file (yes, no) [yes]
  • Promoter lenght, upstream from the TSS (1000, 2000, 5000) [1000]
  • Promoter length, downstream from the TSS (250,500,750,1000) [500]
  • Image width (200...3200) [600]
  • Image height (200...3200) [600]

Details

Based on the input data, a genome-wide coverage is calculated. The resolution for the coverage calculation can be set, and it is by default 50 bps. Smoothing of the resolution is recommended, and can be set separately from the coverage resolution. The default length for smoothing is 400 bp. Normalization is controlled using the setting for fragment length. Users can choose to calculate the enrichment of methylated CpG islands for promoter regions only, and select the length of the promoter region used. The default is 1000 bps upstream and 500 bps downstream from the transcription start site.

During the analysis, the whole genome or just the promoter regions are split into bins of coverage resolution (by default 50 bps). The short read coverage is calculated using this resolution. Since the short reads are not the whole length sonicated DNA fragments, smoothing of the results is usually performed by extending the reads. During the extension, each read is extended to both telomeric and sentromeric directions by the amount set in the smoothing extension parameter. For normalization the local CpG density needs to be estimated. The setting for fragment length specifies the area around each genomic bin that is used for calculating the local CpG density. The fragment length should be set close to the estimated size of the sonicated DNA fragments generated after the amplification.

Several quality control plots are generated:
  • The saturation analysis plot is used for deciding whether the amount of data is enough to cover the genome and generate reliable and reproducible methylation profiles.
  • The coverage analysis plot is used for getting an idea of number of CpGs covered by the short reads, and the depth of the coverage.
  • The calibration curve plot enables examination of the dependency of local CpG densities and signals derived from the data.
The following histograms of the final results are generated:
  • The CpG densities plot features a histogram of the local CpG densities.
  • The RPM signal plot displays a histogram of the number of reads per million of the reads per a genomic bin.
  • The AMS signal plot displays a histogram of absolute methylation scores (a measure normalized by the number of genomic bins covering a certain region of interest in the genome).

Output

Methylation results are given in the file methylation.tsv. Users can also choose to have the relative methylation scores in a BED file, which can be visualized in the Chipster genome browser. Several plots are generated as described above.
