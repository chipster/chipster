Plot a calogram

Description

Plot a calodogram is based on EMBOSS command fdrawgram that is a wrapper command for command drawgram of the PHYLIP package.

This tool rooted phylogenies, cladograms, circular trees and phenograms in PostScript and PDF formats. Similar to DRAWTREE but plots rooted phylogenies.

Parameters

  • Iterate to improve tree (fdrawtree parameter -iterate: )
  • Use branch lengths from user trees (fdrawtree parameter -lengths: )
  • Label direction (fdrawtree parameter -labeldirection: )
  • Angle the tree is to be plotted (fdrawtree parameter -treeangle: )
  • Degrees the arc should occupy (fdrawtree parameter -arc: )
  • Angle of labels (fdrawtree parameter -labelrotation: )
  • Automatically rescale branch lengths (fdrawtree parameter -rescaled: )
  • Centimeters per unit branch length (fdrawtree parameter -bscale: )
  • Depth of tree as fraction of its breadth (fdrawtree parameter -treedepth: )
  • Collect a log file

    • For more details, please check the manual page of fdrawgram command.

    Input and output

    Input is a Phylip formatted tree file.

    Output consists of two plot files containing the same tree. One in PDF and one in PostScript format.
