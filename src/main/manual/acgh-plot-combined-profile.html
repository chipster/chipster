aCGH tools / Plot profiles of matched copy number and expression

Description

Plot profiles of two priorly matched data sets of copy number and expression. This tool must be run on the output from the tool Match copy number and expression probes - matched-cn-and-expression.tsv.

Parameters

  • Samples to be plotted (1...n) [1]
  • Chromosomes to be plotted (0...25) [0]
  • Image width (200...3200) [600]
  • Image height (200...3200) [600]

Details

Plots matched copy number and expression. In the copy number part, log ratios are shown in black, segmented values in blue, probabilities of losses in red (left scale), and probabilities of gains in green (inverted left scale). The expression part contains the original measurements in black and smoothed values in blue.

Output

A plot showing matched copy number and expression data for an individual sample and the chromosome of interest.

References

Wieringen et al. (2009) Nonparametric testing for DNA copy number induced differential mRNA gene expression. Biometrics 65: 19-29
