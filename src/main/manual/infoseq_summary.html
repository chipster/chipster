Sequence file summary

Description

Writes a summary of a sequence set.

Parameters

None

Details

This tool prints a summary report about a given sequence set.

Following properties are listed:

  • Size of the sequence set
  • Sequence file format
  • Sequence type (protein or nucleotide)
  • Number of sequences in the data set
  • Name and the length of the longest sequence (or one of equally long ones)
  • Name and the length of the shortest sequence (or one of equally short ones)
  • Average sequence length
  • Total amount of nucleotides/residues in the data set

    • Output

    Output is a text file.
