Retrieve sequences for a given organism in Ensembl

Description

Retrieves genomes, transcriptomes or proteomes from Ensembl and Ensembl genomes databases.

Parameters

  • Species name This parameter defines the species name or taxonomy code. In species names, spaces should be replaced with underscore character. For example: homo_sapiens or mus_musculus)
  • Sequence data type This parameter defines what sequences are retrieved for the given species: Full genome, transcripts or the protein sequences.
  • List the available species names When this selection is used, Ensemblfetch will retrieve list of species names, available in Ensembl and Ensembl genomes database.

    • Output

    Output is a FASTA formatted sequence file containing the genome, transcriptome or the proteome of the species defined.
