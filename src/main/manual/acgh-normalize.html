Copy number aberrations / Normalize copy number data

Description

Normalizes copy number data (either aCGH log ratios or log2-transformed NGS data).

Parameters

  • Normalization method (median, mode, none) [median]
  • Number of chromosomes (...) [23]

Details

Log ratios are normalized using the chosen method.

The input data must contain these columns: chromosome, start, end.

Normalization can also be done by setting the corresponding parameters in the tool Segment copy number data.

Output

Normalized values.
